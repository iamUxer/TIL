# React

##Class Components
```js
class Hello extens Component {
  render() {
    return <div> Hello, World! </div>
  }
}

```

##Functional Components

```js
// 함수 선언을 이용한 정의
function Hello() {
  return <div> Hello, World! </div>
}

// 함수 표현식
const Hello = () => {
  return <div> Hello, World! </div>
}

```
